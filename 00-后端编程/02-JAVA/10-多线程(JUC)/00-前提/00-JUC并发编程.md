## JUC是什么

JUC 是在java5.0添加的`java.util.concurrent`包的简称, 目的就是为了更好的支持高并发任务.

![](https://youpaiyun.zongqilive.cn/image/20200307102007.png)

## ***JUC可以分为五大类***

```
1、同步工具类

2、lock类

3、原子类

4、集合相关类

5、Executor框架相关类
```
