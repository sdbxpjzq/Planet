## 栈帧

栈帧的概念：java中的方法被扔进虚拟机的栈空间之后就成为“栈帧”，

栈帧中主要保存3类数据:

1. 本地变量(Local variables): 输入和输出参数, 以及方法内的变量
2. 栈操作(Opreand Stack): 记录出栈, 入栈的操作
3. 栈帧数据(Frame Data): 包括类文件, 方法等
