## 写锁

> 写锁会把读和写都堵塞

| session1(写锁)                                  | session2                                       |
| ----------------------------------------------- | ---------------------------------------------- |
| `lock  tables t write`                          |                                                |
| 当前session对锁定表的查询+更新+插入操作都可执行 | 其他session对锁定表的查询被阻塞,需要等待锁释放 |
|                                                 |                                                |
|                                                 |                                                |
|                                                 |                                                |
